# 🚀 TaskMind Lite – Full Stack Task Management App

A full-stack task management web application built using React, Node.js, Express, and MongoDB.  
It allows users to create, update, delete, and manage tasks efficiently with priority levels and completion tracking.

---

## ✨ Features

- ➕ Add new tasks
- ✏️ Edit tasks
- ❌ Delete tasks
- ✔️ Mark tasks as completed / pending
- 📌 Priority levels (High / Medium / Low)
- 🔄 Real-time updates after changes
- 💾 MongoDB database integration

---

## ⚙️ Setup Instructions

### Clone repository
git clone https://github.com/sirivarshini399/taskmind-lite.git

---

### Backend setup
cd backend
npm install
node server.js

Create .env file:
MONGO_URI=your_mongodb_connection_string
PORT=5000

---

### Frontend setup
cd frontend
npm install
npm run dev

---

## 🔗 API Endpoints

GET     /api/tasks        Get all tasks
POST    /api/tasks        Create task
PUT     /api/tasks/:id    Update task
DELETE  /api/tasks/:id    Delete task
